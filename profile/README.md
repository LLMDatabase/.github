> Rethinking databases for the age of large language models.

LLMDatabase explores a new class of data systems where large language models serve as the underlying storage and retrieval substrate. Rather than using LLMs as interfaces to existing databases, we investigate how database abstractions themselves must evolve in the presence of parametric, probabilistic, and semantically generalizing systems.

We study the foundations, system primitives, and design tradeoffs required to build reliable LLM-native data systems.
